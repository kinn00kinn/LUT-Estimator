# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-14

- Split the project into a reusable library package and a CLI.
- Replaced hard-coded file editing with command-line arguments.
- Added `pyproject.toml` for packaging and dependency management.
- Added regression tests for LUT application, LUT estimation endpoints, and `.cube` export.
- Rewrote the main README in English and added a separate Japanese README.
- Added contributing guidance and GitHub Actions test workflow.
