# Code of Conduct

## Our Standard

This project expects respectful, constructive collaboration. Harassment, insults, and bad-faith behavior are not acceptable.

## Expected Behavior

- Assume good intent and communicate clearly.
- Keep technical disagreement focused on the work.
- Be respectful of different experience levels and backgrounds.

## Unacceptable Behavior

- Personal attacks or harassment
- Discriminatory language
- Repeated disruptive behavior in issues or pull requests

## Enforcement

Project maintainers may moderate discussions, close threads, or block contributions that violate this code of conduct.
